qrenderer.RenderDocMembersMixin

RenderDocMembersMixin(layout_obj, renderer, level=1)

Extend Rendering of objects that have members

Parameter Attributes

layout_obj: (
    layout.DocClass
    | layout.DocFunction
    | layout.DocAttribute
    | layout.DocModule
    | layout.Page
    | layout.Section
    | layout.Link
    | layout.Layout
)

Layout object to be documented

renderer: QRenderer

Renderer that holds the configured values

level: int = 1

The deepth of the object in the documentation

show_description

show_description : bool = field(init=False, default=True)

Whether to show the description of the object

This only applies to objects that have descriptions that are not considered part of the body documentation body.

This attribute is not well defined and it may change in the future.

show_object_symbol

show_signature

show_signature : bool = field(init=False, default=True)

Whether to show the signature

show_signature_annotation

show_signature_annotation : bool = field(init=False, default=False)

Where to show type annotations in the signature

The default is False because they are better displayed in the parameter definitions.

show_signature_name

show_signature_name : bool = field(init=False, default=True)

Whether to show the name of the object in the signature

show_title

show_title : bool = field(init=False, default=True)

Whether to show the title of the object

The title includes:

1. symbol for the object
2. name of the object
3. labels the object

Each of this can be independently turned off.
